Why test?

1. We are not perfect

2. Computers are great at automation

3. Codex says so

When you're ready to release a new version of the Plugin... Make sure everything is committed and the new version actually works. Pay attention to all WordPress versions your Plugin supports and try to test it with all of them. Don't just test the new features; also make sure you didn't accidentally break some older functionality of the Plugin.
codex.wordpress.org/Writing_a_Plugin
